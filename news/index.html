Dunedin VR trip
-7.18.2021-

This past week we had the awesome oppertunity to go along to the Dunedin Game Developers meetup.
The theme for the night was everything VR, with around 10 VR rigs set up around the place we had a good go at a few different games.

Highlight game for myself (Nat here) would have to have been the game "Superhot". Which if you have not heard of it, is a fast paced action game where you dodge bullets like the matrix and feel like a "badass"!
big thanks to the guys @code_nz for putting on a great VR event lastnight, it was well worth the drive from Invercargill!
